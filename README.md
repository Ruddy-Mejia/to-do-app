
# To do App

Generic to-do list, responsive and modern design with TailwindCSS, use it as you like :D


## Author

## Install

To deploy this project run


```bash
  npm install
  ng serve
  cd .\backend\
  npm install
  node server.js
```

##   Requirements 
- node.js 22.12 or higher
- npm 10.9 or higher
